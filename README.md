## ⭐ Tipsy-iOS App ⭐

An expense tracker app to manage group expenses and bills. Best for millenials as it tracks your expenses when in pairs or groups. Built using Swift and the SwiftUI Framework.

## 📌 Tech Stack

Swift, UIKit, Z-Shell, XCode
